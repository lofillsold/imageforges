ImageForge

Профессиональный инструмент для работы с изображениями прямо в вашем браузере. Конвертация, управление метаданными и умное сжатие — всё без загрузки файлов на сервер.

Полная приватность

Все операции выполняются локально в вашем браузере. Файлы никуда не отправляются — ни на сервер, ни в облако. Ваши изображения остаются только у вас.

Высокая скорость

Благодаря локальной обработке и современным WebAssembly-кодекам, конвертация происходит мгновенно. Никаких ожиданий загрузки и выгрузки файлов.

Метаданные

Просматривайте и редактируйте EXIF, IPTC, XMP данные: модель камеры, параметры съёмки, GPS-координаты, автор и копирайт. Удаляйте конфиденциальную информацию в один клик.

Умное сжатие

Оптимальное соотношение размера и качества. Наглядное сравнение оригинала и сжатой версии с помощью интерактивного ползунка — двигайте его, чтобы увидеть разницу.

Как это работает

1

Загрузка

Файл остаётся в памяти вашего браузера. JavaScript получает доступ к данным через File API.

2

Декодирование

Специализированные WASM-библиотеки декодируют изображение в пиксельные данные (RGBA).

3

Обработка

Canvas API применяет нужные операции: изменение формата, сжатие, удаление метаданных.

4

Результат

Готовый файл создаётся через Blob API и скачивается напрямую на ваше устройство.

Технологии под капотом: Canvas API для базовой обработки, WebAssembly-кодеки @jsquash/avif и @jsquash/jxl для современных форматов, UTIF.js для TIFF, heic2any для HEIC, ag-psd для Photoshop, exifr и piexifjs для работы с метаданными. Всё работает в любом современном браузере без установки расширений.
